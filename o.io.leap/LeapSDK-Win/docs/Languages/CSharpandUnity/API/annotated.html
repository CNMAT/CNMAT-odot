Data Structures
Here are the data structures with brief descriptions:

Leap
CircleGestureThe CircleGesture classes represents a circular finger movement
ConfigThe Config class provides access to Leap Motion system configuration information
ControllerThe Controller class is your main interface to the Leap Motion Controller
DeviceThe Device class represents a physically connected device
DeviceListThe DeviceList class represents a list of Device objects
FingerThe Finger class represents a tracked finger
FingerListThe FingerList class represents a list of Finger objects
FrameThe Frame class represents a set of hand and finger tracking data detected in a single frame
GestureThe Gesture class represents a recognized movement by the user
GestureListThe GestureList class represents a list of Gesture objects
HandThe Hand class reports the physical characteristics of a detected hand
HandListThe HandList class represents a list of Hand objects
InteractionBoxThe InteractionBox class represents a box-shaped region completely within the field of view of the Leap Motion controller
KeyTapGestureThe KeyTapGesture class represents a tapping gesture by a finger or tool
ListenerThe Listener class defines a set of callback functions that you can override in a subclass to respond to events dispatched by the Controller object
MatrixThe Matrix struct represents a transformation matrix
PointableThe Pointable class reports the physical characteristics of a detected finger or tool
PointableListThe PointableList class represents a list of Pointable objects
ScreenThe Screen class represents a computer monitor screen
ScreenListThe ScreenList class represents a list of Screen objects
ScreenTapGestureThe ScreenTapGesture class represents a tapping gesture by a finger or tool
SwipeGestureThe SwipeGesture class represents a swiping motion of a finger or tool
ToolThe Tool class represents a tracked tool
ToolListThe ToolList class represents a list of Tool objects
VectorThe Vector struct represents a three-component mathematical vector or point such as a direction or position in three-dimensional space
